# Roadmap

## Etapa 1 - Fundacao do MVP

- Estruturar API.
- Definir modelo de perfil do usuario.
- Definir modelo de vaga.
- Calcular compatibilidade inicial.
- Gerar CV PDF simples.

## Etapa 2 - Experiencias reutilizaveis

- Separar experiencias em blocos.
- Permitir diferentes enfoques para a mesma experiencia.
- Selecionar blocos mais relevantes para cada vaga.

## Etapa 3 - Banco de vagas e ranking diario

- Gerar links de busca por cargo, localidade, modalidade e skills. Concluido na base inicial.
- Importar rascunho de vaga por URL/HTML para revisao humana. Concluido na base inicial.
- Salvar vagas analisadas. Concluido na base inicial.
- Salvar perfis, candidaturas e PDFs gerados. Concluido na base inicial.
- Detectar duplicatas.
- Ranqueamento por compatibilidade. Concluido na base inicial.
- Fila diaria de candidaturas recomendadas. Base iniciada com endpoint de recomendacoes.

## Etapa 4 - Revisao humana e candidatura assistida

- Painel de aprovacao. Base iniciada com interface web em `/`.
- Mensagem personalizada para cada vaga.
- Checklist antes do envio.
- Registro de status da candidatura. Base iniciada com atualizacao de status.

## Etapa 5 - Automacao responsavel

- Integracoes permitidas por API ou fluxo assistido.
- Preenchimento semi-automatico com confirmacao do usuario.
- Limites diarios configuraveis.
- Logs do que foi enviado.
